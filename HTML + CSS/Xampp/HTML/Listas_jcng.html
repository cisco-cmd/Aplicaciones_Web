Listas ordenadas y desordenadas

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.

Hay tres tipos de listas:

  • Listas ordenadas (ol)
  • Listas desordenadas (ul)
  • Listas de definiciones (dl)

En este caso, vamos a utilizar solo listas ordenadas y desordenadas las cuales se introducen con las etiquetas ol y ul, para introducir cada punto dentro de ellas se utiliza la etiqueta li.
