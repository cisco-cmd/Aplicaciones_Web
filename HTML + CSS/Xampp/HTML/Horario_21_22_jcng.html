Una tabla en HTML consiste en un conjunto de celdas dentro de filas y columnas. En este caso haremos dos horarios, uno poniendo la asignatura en su celda correspondiente y otra poniendo uniendo las celdas de cada asignatura según el horario que tengan (rowspan/colspan) Para hacer la tabla utilizamos "table", la creación de las celdas va de derecha a izquierda, para crear una fila utilizamos "tr" y para crear cada una de las celdas se hace con "td y th"
