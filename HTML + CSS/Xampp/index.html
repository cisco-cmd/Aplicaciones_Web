HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript). En este conjunto de páginas HTML aprenderás como insertar en HTML los títulos y los tipos que hay, los tipos de parrafos, estilos de letras, hiperenlaces, imagenes, tablas, listas, formularios, etc. En la parte superior, puedes acceder a las diferentes páginas.
